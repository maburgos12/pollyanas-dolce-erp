De dónde sale la información del SAT

1. e.firma de la empresa

El ERP usa el certificado, la llave privada y la contraseña fiscal configurados en producción. Con eso pide acceso al servicio oficial de Descarga Masiva del SAT.

2. Descarga de XML

El proceso solicita CFDI emitidos y recibidos por periodo. Cuando el SAT termina la solicitud, devuelve paquetes ZIP con los XML fiscales.

3. Datos para conciliación

De cada XML se guarda UUID, RFC emisor, RFC receptor, total, moneda, fecha, método de pago, forma de pago y el XML original para auditoría.
