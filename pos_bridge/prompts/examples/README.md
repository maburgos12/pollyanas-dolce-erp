Este directorio guarda ejemplos curados de preguntas reales de Direccion General.

Objetivo:
- convertir errores reales de respuesta en reglas reutilizables
- mostrar patron malo vs patron bueno
- reforzar criterio ejecutivo sin perder trazabilidad ERP

Uso recomendado:
- cuando la IA falle, extraer solo el caso estable
- guardar pregunta, error, respuesta correcta y regla aprendida
- si la regla aplica siempre, tambien subirla a `memory.md`
